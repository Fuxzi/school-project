# ✨ Additional Features
---
* **User Authentication**

  * Secure login & registration system
  * Passwords hashed using **bcrypt**
  * Role-based access (**admin** & **user**)

* **User Profile Management**

  * View and update profile information (name & email)
  * Change password from settings page
  * Personalized dropdown profile menu

* **Review System**

  * Users can submit product reviews with ratings
  * Edit or delete their own reviews
  * Dedicated **“My Reviews”** page for easy management

* **Product Search**

  * Search products by title, type, or description
  * Clean search results with images, categories, and tags
  * Automatic fallback image if product image is missing

* **Admin Panel**

  * Manage products and categories
  * Assign or revoke admin roles
  * Delete users with self-protection (admin cannot delete own account)

* **Security Enhancements**

  * CSRF protection on forms and sensitive actions
  * Server-side validation for inputs
  * Safe database queries using prepared statements

* **Modern UI**

  * Glassmorphism (glass UI) design
  * Responsive layout for desktop and mobile
  * Consistent theme across site, admin, and auth pages

* **Non-Ecommerce Focus**

  * No cart, checkout, coupons, or purchase history
  * Website is purely focused on **product reviews & opinions**
